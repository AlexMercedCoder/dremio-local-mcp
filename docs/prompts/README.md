# Dremio MCP Prompts Reference

Prompts allow you to trigger complex, multi-step workflows with a single command. The AI Agent uses these "templates" to know exactly how to perform high-value tasks.

## Analysis & Documentation
- [analyze_dataset](analyze_dataset.md): Comprehensive health check and status report.
- [draft_wiki](draft_wiki.md): Auto-generate business documentation.
- [generate_view](generate_view.md): Architect semantic views (Silver/Gold layers).
- [onboard_analyst](onboard_analyst.md): Guide for new users exploring a Space.
- [compare_schemas](compare_schemas.md): Diff two datasets (Dev vs Prod).

## Optimization & Operations
- [troubleshoot_job](troubleshoot_job.md): Diagnose failures and suggest fixes.
- [optimize_query](optimize_query.md): Rewrite SQL for performance.
- [suggest_reflections](suggest_reflections.md): Propose acceleration reflections.
- [audit_users](audit_users.md): See who is accessing data.
- [find_unused_assets](find_unused_assets.md): Identify "zombie" data for cleanup.
